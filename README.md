## Installation

Install my-project with npm

```bash
    npm install -g surge-explorer
```

Run with
```bash
    surge-explorer
```
or use npx directly
```bash
    npx surge-explorer
```

After running the command you will see a prompt like this

```bash
Go on localhost:3000 to open the explorer 
Press Ctrl+C to close it
```
